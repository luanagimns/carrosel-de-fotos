# 📸 Photo Carousel

Um carrossel de imagens desenvolvido com **HTML**, **CSS** e **JavaScript puro**.

O projeto permite navegar entre imagens utilizando botões de avançar e voltar. O carrossel não é infinito: ao chegar na primeira ou na última imagem, o botão correspondente é desabilitado automaticamente.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## ✨ Funcionalidades

- 📷 Exibição de imagens em carrossel
- ⬅️ Botão para voltar
- ➡️ Botão para avançar
- 🔘 Indicadores da imagem atual (bolinhas)
- 🚫 Botões desabilitados na primeira e na última imagem
- 📱 Layout responsivo

---

## 📚 Conceitos Praticados

Durante o desenvolvimento deste projeto foram praticados conceitos importantes de Front-end, como:

- Estruturação de páginas com HTML
- Estilização com CSS
- Flexbox
- Manipulação do DOM
- Seleção de elementos
- Eventos de clique
- Manipulação de classes CSS
- Controle de estado utilizando variáveis
- Estruturas de repetição (`forEach`)
- Organização de arquivos
